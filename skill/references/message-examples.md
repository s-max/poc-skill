# Client Message Examples

Match the meeting's language (Dutch/English) and tone (formal/informal).

## Informal Dutch

```
Hey [Name],

Naar aanleiding van ons gesprek over [specific topic] - en de uitdaging dat [specific pain point] - heb ik een eerste versie gemaakt.

Hier kun je hem bekijken: [URL]

Ik heb het opgezet zoals jullie het bij [internal reference] doen, maar dan toegespitst op [their context]. Zou handig zijn om dit te hebben voor [agreed timeline/meeting].

Stuur gerust [requested materials] door, dan kan ik dat er nog in verwerken voor [timeframe].

Groet,
Max
```

## Formal English

```
Hi [Name],

Following our discussion about [specific topic] - particularly [specific pain point] - I've put together an initial prototype.

You can view it here: [URL]

I've modeled it on [internal reference], adapted for [their context]. This could be useful to have ready before [agreed timeline/meeting].

Once you send over [requested materials], I can incorporate those details before [timeframe].

Best regards,
Max
```

## Key Elements

1. **Open with specific callback** - Reference exact topic/pain from meeting
2. **Include live URL** - The POC link
3. **Connect to their context** - Use their internal terminology
4. **Reference agreed timeline** - Show you listened
5. **Clear next step** - What you need from them
